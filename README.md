# Data Analyst Test Project

## Описание проекта
Решения тестового задания по вероятности, Python, SQL, статистике/A-B и базовому ML. Проект оформлен как готовый репозиторий

## Стек технологий
Python 3.10+, SQL (PostgreSQL/SQLite-совместимый синтаксис), Markdown

## Структура
```
project-name/
├── README.md
├── probability/solutions.md
├── python/isomorphic.py
├── python/missing_number.py
├── python/prime_factors.py
├── sql/applicants.sql
├── sql/full_join.md
├── sql/purchases.sql
├── statistics_ab/solutions.md
└── ml_base/solutions.md
```

## Краткое описание блоков
`probability/` — задачи по вероятности и логике  
`python/` — алгоритмические задачи на Python с оценкой сложности  
`sql/` — запросы и пояснения по SQL  
`statistics_ab/` — статистика и A/B тесты  
`ml_base/` — краткие ответы на вопросы по ML

## Как запустить Python-скрипты
Из корня проекта:
```
python python/isomorphic.py --s paper --t title
python python/missing_number.py --nums 1 2 3 4 5 6 8 9 10 11
python python/prime_factors.py --n 56
```
